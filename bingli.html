一、名词解释

细胞与组织损伤与修复
1化生

一种分化成熟的组织因受刺激因素的作用而转化为另一种分化成熟组织的过程。

2玻璃样变性

在细胞或间质内出现红染、均质、半透明的蛋白性物质的蓄积。

3坏疽

较大面积坏死并伴有不同程度腐败菌感染，使坏死组织呈黑褐色者称为坏疽。

4肉芽组织

即增生旺盛的幼稚结缔组织，主要由新生毛细血管和成纤维细胞构成，周围有炎细胞浸润。

血液循环障碍
5静脉性充血

由于静脉回流受阻，血液淤积于小静脉和毛细血管内，引起局部组织、器官血管内血量增多。又称被动性充血，简称淤血。

6心力衰竭细胞

吞噬有含铁血黄素的巨噬细胞。

7血栓形成

在活体的心脏或血管内血液有形成分形成固体质块的过程。

8栓塞

循环血液中出现的不溶于血液的异常物质，随血液流动，阻塞管腔的现象。

9梗死

局部组织、器官由于血流供应迅速中断而引起的缺血性坏死。

29弥漫性血管内凝血（DIC）

在某些致病因素作用下，机体凝血系统被激活，大量可溶性促凝物质入血而引起的以凝血功能障碍为主要特征的病理过程。

炎症
10炎症

具有血管系统的活体组织对各种原因损伤因子的刺激所发生的一种以防御为主的基本病理过程。

11炎症介质

又称化学介质，是细胞崩解或体液中产生的一类具有血管活性作用的物质，又称血管活性物质。

12蜂窝织炎

疏松结缔组织由溶血性链球菌引起的弥漫性的化脓性炎症。

13脓肿

由金黄色葡萄球菌引起的局限性化脓性炎症。

肿瘤
14癌

源于上皮组织的恶性肿瘤。

15肉瘤

来源于间叶组织的恶性肿瘤。

16异型性

肿瘤组织与正常组织在细胞形态和组织结构上的差异，仅指肿瘤的实质。

17恶病质

恶性肿瘤晚期患者可发生严重消瘦、无力、贫血、全身衰竭，皮肤干枯呈黄褐色。

18癌前病变

某些具有癌变倾向的良性病变，如长期未治愈即可能转变为癌。

19角化珠 / 癌珠

分化好的鳞癌形成的癌巢中央有同心圆状角化物。

20原位癌

癌变仅限于上皮层而未突破基膜。

21浸润性癌

原位癌继续发展，突破基膜而浸润至真皮或黏膜下层。

心血管疾病
22心绞痛

心脏急性暂时性缺血缺氧所致的以胸痛为特点的临床综合征。

23心肌梗死

心肌由于严重持续性缺氧而引起的坏死。

24风湿病

与A族乙型溶血性链球菌感染有关的超敏反应性疾病，病变主要累及全身结缔组织。

呼吸系统
25大叶性肺炎

主要由肺炎链球菌引起的以肺泡内弥漫性纤维素渗出为主的炎症。

26肺肉质变

大量纤维素被肉芽组织取代而发生机化，使病变肺组织呈褐色肉样。

27小叶性肺炎

主要由化脓菌引起，以肺小叶为单位的急性化脓性炎症性疾病。

28慢性阻塞性肺疾病（COPD）

一组慢性肺实质及小气道受损引起慢性气道阻塞、呼吸阻力增加和肺功能不全为共同特征的肺疾病统称。

二、知识点整理

95条
病理学基础
  1. 最常用的病理学研究方法是活体组织检查。
  2. 病理过程分类：
    局部：血栓形成、栓塞、梗死、炎症
    全身：发热、休克、弥散性血管内凝血
  3. 病理状态举例：烧伤后的皮肤瘢痕、关节炎后的关节强直。
  4. 疾病发生原因中最常见的是生物性因素。
  5. 适应在形态上表现为萎缩、肥大、增生、化生。
细胞适应与损伤
  1. 萎缩类型：
    类型举例
    营养不良性动脉粥样硬化、肾缺血、食道癌
    压迫性肾盂积水、脑积水
    失用性骨折后肢体长期固定→废用性萎缩
    神经损伤性脊髓灰质炎→肌肉萎缩
    内分泌性卵巢切除→子宫萎缩
    其他鼻咽癌→涎腺萎缩
  2. 萎缩病理变化中电镜下可见脂褐素。
  3. 化生类型：
    类型过程
    鳞状上皮化生柱状上皮（子宫颈管、支气管腺上皮）/ 移行上皮 → 鳞状上皮
    肠上皮化生萎缩性胃炎时胃黏膜上皮
    结缔组织化生骨化性肌炎→形成骨组织
  4. 细胞水肿好发部位：心、肝、肾。
  5. 脂肪变性好发部位：心、肝、肾，肝最常见。
  6. 心肌脂肪变性可出现虎斑心。
  7. 血管玻璃样变性常见于高血压病时的全身细动脉。
坏死与修复
  1. 细胞核的变化是细胞坏死的主要形态学标志：表现为核固缩、核碎裂、核溶解。
  2. 坏死类型：
    类型特点部位 / 举例
    凝固性坏死组织灰白、干燥、质实心、肾、脾
    液化性坏死组织液化最常见于脑组织（脑软化）
    坏疽见下表—
    纤维素样坏死结缔组织小血管壁风湿病、恶性高血压
    坏疽分型对比 ↓
    分型部位特点
    干性四肢（下肢远端）与健康组织界限分明
    湿性与外界相通的内脏（肺、肠、子宫、阑尾）界限不明显，可引起全身中毒
    气性肌肉深部开放性创伤合并产气荚膜杆菌感染
  3. 细胞再生能力分类：
    类型细胞
    不稳定细胞表皮细胞、淋巴细胞、造血细胞、黏膜及腺体上皮
    稳定细胞肝、肾、胰腺、涎腺、皮脂腺；间叶组织（成纤维细胞、骨母细胞、平滑肌细胞、内皮细胞等）
    永久性细胞中枢神经细胞、周围神经节细胞、骨骼肌、心肌细胞
  4. 肉芽组织功能：①抗感染及保护创面；②填补创口和组织缺损；③机化坏死组织、凝血块和异物。
  5. 骨折愈合过程：血肿形成 → 纤维性骨痂 → 骨性骨痂 → 骨痂改建/再塑。
血液循环障碍
  1. 左心衰 → 肺淤血，镜下可见心衰细胞。
  2. 慢性肝淤血常见于右心衰，可形成槟榔肝，后发展为淤血性肝硬化。
  3. 血栓形成条件：①心血管内皮细胞损伤；②血流状态改变；③血液凝固性增加。
  4. 血小板活化在血栓形成中起关键作用，包括：粘附、释放、黏集三个反应。
  5. 延续性血栓形成顺序：白色血栓 → 混合性血栓 → 红色血栓。
  6. 血栓类型：白色血栓、混合性血栓、红色血栓、透明血栓。
  7. 栓子运行方向为顺血流方向。
  8. 栓塞规律：
    右心/体静脉 → 肺动脉系统
    左心/肺静脉/体循环动脉 → 脑、脾、肾、下肢
    门静脉系统 → 肝内门静脉分支
  9. 血栓栓塞为最常见栓塞类型。肺动脉栓塞约95%血栓栓子来自下肢深部静脉（腘静脉、股静脉、髂静脉）。
  10. 特殊栓塞：
    氮气栓塞（沉箱病/减压病）→ 潜水员
    脂肪栓塞 → 长骨粉碎性骨折 / 严重脂肪组织挫伤
    羊水栓塞 → 分娩
  11. 梗死类型：
    类型坏死性质部位形状
    贫血性梗死凝固性坏死心、肾、脑、脾脾肾：锥体形；心：不规则/地图形；脑：不规则状
    出血性梗死液化性坏死肺、肠肺：锥体形；肠：节段状
炎症
  1. 炎症基本病变：变质、渗出、增生。
  2. 白细胞渗出过程：附壁 → 黏附 → 游出 → 趋化作用。
  3. 形态学分类：变质性炎、渗出性炎、增生性炎。
  4. 变质性炎多由病毒或毒素引起，如：病毒性肝炎、乙型脑炎、脊髓灰质炎、白喉外毒素→心肌炎、伤寒内毒素→肌肉蜡样坏死。
  5. 渗出性炎分类：
    类型好发部位举例
    浆液性炎浆膜、黏膜、疏松结缔组织—
    纤维素性炎浆膜、黏膜、肺组织绒毛心；咽/气管白喉；菌痢；大叶性肺炎
    化脓性炎—蜂窝织炎、脓肿、表面化脓和积脓
    出血性炎——
  6. 特异性增生性炎中，肉芽肿形成是由于巨噬细胞增生。
肿瘤
  1. 肿瘤最基本生物学特征：无限增殖与分化异常。
  2. 异型性与分化程度：
    良性：分化高 → 异型性小
    恶性：分化低 → 异型性大
    口诀：分化高→异型小→恶性低；分化低→异型大→恶性高
  3. 生长方式：膨胀性生长、浸润性生长、外生性生长。
  4. 转移方式：
    淋巴道转移 — 癌常见
    血道转移 — 肉瘤常见
    种植性转移
  5. 胃癌 → 可逆流转移至左锁骨上淋巴结。
  6. 血源性转移瘤形态：多个散在、球形、边界清楚。
  7. 常见癌前病变：黏膜白斑、纤维囊性乳腺病、结直肠腺瘤、慢性萎缩性胃炎、胃溃疡。
心血管疾病
  1. 动脉粥样硬化：血脂异常 → 脂质沉积于大中动脉内膜 → 粥样斑块。
  2. 低密度脂蛋白（LDL）与动脉粥样硬化和冠心病的发生极为密切。
  3. 粥样斑块继发变化/并发症：斑块内出血、斑块溃疡、血栓形成、钙化、动脉瘤形成（最危险最严重）。
  4. 主动脉粥样硬化严重程度：腹主动脉 > 胸主动脉 > 主动脉弓 > 升主动脉。
  5. 冠状动脉粥样硬化最常发生于左冠状动脉前降支，其次为右冠状动脉干、左冠状动脉旋支。
  6. 冠心病类型：隐匿性心肌缺血、心绞痛、心肌梗死、心肌硬化、冠状动脉性猝死。
  7. 心肌梗死部位：
    供血动脉比例梗死部位
    左冠状动脉前降支40%~50%左心室前壁、心尖部、室间隔前2/3
    右冠状动脉30%~40%左心室后壁、室间隔后1/3、右心室大部
    左冠状动脉旋支15%~20%左心室侧壁
  8. 高血压病基本病变特征：全身细小动脉硬化。
  9. 向心性肥大：左心室代偿性肥大 → 出现牛心。
  10. 脑软化好发：基底节、丘脑、脑桥、小脑。
  11. 脑出血是高血压病最严重的并发症，常见于基底节、内囊。
  12. 风湿病分期：变质渗出期 → 增生期（肉芽肿期） → 纤维化期。
  13. 风湿性心内膜炎主要侵犯心瓣膜，二尖瓣最常见。
  14. 风湿性心包炎 → 绒毛心，听诊可闻及心包摩擦音。
呼吸系统
  1. 大叶性肺炎好发：单侧，左/右肺下叶多见。
  2. 小叶性肺炎好发：双肺下叶；特征：以细支气管为中心的化脓性炎 / 以肺小叶为单位的急性化脓性炎。
  3. 病毒包涵体是诊断病毒性肝炎的重要依据。
  4. 慢性肺源性心脏病：心脏病变以右心室为主，出现横位心。
结核病
  1. 结核病原菌对人致病的主要是人型和牛型。
  2. 典型结核结节：中央干酪样坏死 + 大量上皮样细胞 + 一些朗汉斯巨细胞 + 外周淋巴细胞 + 少量成纤维细胞。
  3. 原发综合征：肺部原发病灶 + 结核性淋巴管炎 + 肺门淋巴结结核，X线呈哑铃状阴影。
  4. 原发性肺结核好发：肺上叶下部/肺下叶上部，靠近胸膜，右肺多见。
  5. 继发性肺结核好发：肺尖部，右肺多见。
  6. 浸润性肺结核是继发性肺结核最常见的临床类型。
  7. 肠结核溃疡与肠管长轴垂直，引起肠腔狭窄。
    对比：肠伤寒溃疡与肠管长轴平行，引起穿孔/出血。
消化系统
  1. 慢性胃炎最常见病因：幽门螺杆菌（Hp）感染。
  2. 慢性萎缩性胃炎光镜下可见肠上皮化生。
  3. 胃溃疡发生机制：
    胃溃疡：迷走神经功能↓ → 胃蠕动↓ → 食物潴留 → 胃窦G细胞 → 胃泌素↑ → 胃酸↑ → 溃疡
    十二指肠溃疡：迷走神经功能↑ → 胃液分泌↑↑ → 溃疡
  4. 消化性溃疡并发症：出血（多见）、穿孔、幽门梗阻、癌变。
  5. 病毒性肝炎性质：变质性炎。
  6. 传播途径：甲肝 → 消化道/粪-口；乙肝 → 血液传播。
  7. 乙型肝炎病毒是我国门脉性肝硬化的主要原因。
  8. 门脉高压症表现：脾肿大、消化功能不良、腹水、侧支循环建立。
  9. 雌激素增多 → 蜘蛛痣和肝掌。
泌尿系统
  1. 肾小球毛细血管丛周边部：毛细血管内皮细胞 + 基膜 + 肾球囊脏层上皮细胞。
  2. 肾病综合征（三高一低）：高蛋白尿、低蛋白血症、高度水肿、高脂血症及脂性尿。
  3. 毛细血管内增生性肾小球肾炎 = 急性肾炎 = 链球菌感染后肾炎 = 大红肾/蚤咬肾，特征：内皮细胞和系膜细胞增生。
  4. 膜性肾小球病 = 大白肾，特征：基膜弥漫性增厚，银染色见钉突。
  5. 膜性肾小球病是引起肾病综合征常见原因之一。
  6. 慢性硬化性肾小球肾炎：特征为肾小球硬化、纤维化，出现玻璃样变性。
  7. 急性肾盂肾炎感染途径：
    途径病原菌备注
    上行性感染大肠杆菌常见类型
    下行性（血源性）感染金黄色葡萄球菌—
传染病
  1. 伤寒：全身单核巨噬细胞系统的急性增生性炎。
  2. 细菌性痢疾：纤维素性炎，部位在乙状结肠和直肠。过程：卡他性炎 → 假膜性炎及溃疡（地图状）→ 愈合。
  3. 流脑：脑膜炎奈瑟菌 → 急性呼吸道传染病 → 脑脊髓膜急性化脓性炎。
  4. 乙脑：乙型脑炎病毒 → 中枢神经系统急性传染病 → 中枢神经系统变质性炎。
DIC、休克与其它
  1. DIC：血液由高凝转向低凝 → 出血倾向。临床表现：出血、休克、器官功能障碍、溶血性贫血。
  2. 休克最易发生类型：感染性休克（内毒素性）；青霉素过敏 → 过敏性休克。
  3. 微循环障碍是休克发生发展的共同发病环节。
  4. 休克时肾脏是最早受损害的器官。
  5. 假性神经递质：苯乙醇胺、羟苯乙醇胺
    正常神经递质：去甲肾上腺素、多巴胺

三、简答题

9题 · 点击展开
▶ 1. 肉芽组织的病理变化特点

肉眼：

呈鲜红色，颗粒状，质地柔软，似鲜嫩肉芽，触之易出血，无痛觉。

镜下：

  1. 向创面垂直生长，以小动脉为轴心，形成襻状弯曲的毛细血管网；
  2. 新生毛细血管之间有大量成纤维细胞；
  3. 肉芽组织内常有不同程度炎细胞浸润。
▶ 2. 渗出液的作用

利：

  1. 防御：稀释、中和毒素，运走毒性代谢产物；
  2. 杀菌：带来抗体和补体，杀灭病原微生物；
  3. 防扩散：纤维素网网罗细菌，局限炎症，防止蔓延；纤维素支架和调理素有利于表面吞噬作用。

弊：

  1. 压迫：心包腔及胸膜腔渗出液过多 → 妨碍心肺活动；
  2. 粘连：过多纤维素发生机化 → 引起器官粘连。
▶ 3. 炎症介质的主要作用
  1. 扩张细动脉和细静脉，血管壁通透性增高；
  2. 有致癌和致热的生物活性；
  3. 白细胞趋化作用。
▶ 4. 动脉粥样硬化的基本病理变化

① 脂纹及脂斑

肉眼：动脉内膜面可见黄色针头帽大小的斑点或宽约1~2mm的淡黄色条纹，平坦或微隆起。

镜下：内膜下大量泡沫细胞聚集（大部分巨噬细胞来源，少部分平滑肌细胞来源）；SMC及较多细胞外基质；少量胆固醇、WBC。

② 纤维斑块

肉眼：内膜表面隆起的灰黄色纤维斑块。

镜下：表面纤维帽，其下为增生的平滑肌细胞、巨噬细胞、泡沫细胞，以及细胞外脂质和基质及炎细胞。

③ 粥样斑块

肉眼：内膜面明显隆起的灰黄色斑块，切面表层白色质硬，深层黄色/黄白色质软粥样物质。

镜下：表层纤维帽；深层大量无定形坏死崩解产物、胆固醇结晶和钙盐沉积；可见肉芽组织、泡沫细胞和淋巴细胞；中膜因压迫变薄。

▶ 5. 大叶性肺炎各期病理变化

① 充血水肿期

肉眼肺叶肿胀，重量增加，呈暗红色，切面湿润可挤出血性浆液
镜下毛细血管扩张充血，肺泡腔内有浆液、少量红细胞、中性粒细胞、巨噬细胞，可检出肺炎链球菌
痰粉红色泡沫样痰

② 红色肝样变期

肉眼肺叶肿胀，色暗红，质实如肝，切面颗粒状
镜下大量纤维素、红细胞及一些中性粒细胞、巨噬细胞渗出，可检出肺炎链球菌
痰铁锈色痰

③ 灰色肝样变期

肉眼充血消退，由暗红转为灰白色，质实如肝
镜下纤维素增多，压迫毛细血管致贫血；大量中性粒细胞和巨噬细胞；已无病菌
痰黄脓痰

④ 溶解消散期

肉眼肺组织质地变软，颗粒状消失，脓样浑浊液流出
镜下脓细胞崩解释放蛋白水解酶 → 纤维素溶解（咳出/淋巴管吸收/巨噬细胞吞噬）→ 肺泡净化
痰稀薄脓性痰
▶ 6. 胃溃疡的病理变化

肉眼：

  1. 多发生在胃小弯，愈近幽门处愈多见，多为单发；
  2. 呈圆形或椭圆形，直径一般在2cm以内；
  3. 溃疡边缘整齐如刀割状；周围黏膜皱襞呈以溃疡为中心的星芒状；切面有时呈斜漏斗状，尖端指向贲门。

镜下（由内到外四层结构）：

  1. 渗出层：少量炎性渗出物（中性粒细胞、纤维素）覆盖溃疡底表面；
  2. 坏死层：无结构红染的坏死组织，有炎细胞浸润；
  3. 肉芽组织层：新生毛细血管和成纤维细胞，有炎细胞浸润；
  4. 瘢痕层：大量胶原纤维和少数纤维细胞。
▶ 7. 休克三期的微循环变化特点

① 微循环缺血期（休克早期 / 代偿期）

特点：少灌少流，灌少于流

皮肤、内脏的微动脉、后微动脉、毛细血管前括约肌和微静脉、小静脉均持续痉挛，后微动脉和毛细血管前括约肌收缩更显著 → 前阻力↑↑ → 真毛细血管网大量关闭 → 动静脉吻合支开放 → 微循环灌流急剧减少 → 组织缺血缺氧。

② 微循环淤血期（休克期 / 失代偿期）

特点：灌多流少

微动脉、后微动脉及前括约肌由收缩转为舒张 → 前阻力↓ → 而微静脉仍收缩 → 后阻力↑ → 大量血液淤积微循环 → 回心血量急剧减少 → 动脉血压显著下降。

③ 微循环凝血期（DIC期 / 休克难治期）

特点：不灌不流

微循环内广泛纤维蛋白（透明）血栓形成，常有局灶性或弥漫性出血 → 组织严重缺氧酸中毒 → 变性坏死 → 多脏器功能障碍（衰竭）→ 动脉血压显著下降。

▶ 8. 炎症的并发症
  1. 转移性脓肿：脓液穿破脓膜，顺着组织间隙流注，或化脓菌进入血流，引起他处脓肿。
  2. 窦道：深部组织的脓肿向外穿通体表或体腔，形成一个向外排脓的盲端管道。
  3. 瘘管：脓肿同时向内外穿通（向内穿通自然管道，向外穿通体表），形成两个以上开口的通道。
  4. 糜烂：表浅化脓 → 坏死组织脱落 → 基膜以上的局限缺损。
  5. 溃疡：表浅化脓 → 坏死组织脱落 → 超过基膜的缺损。
  6. 疖：金黄色葡萄球菌引起的单个毛囊、皮脂腺及其附近组织的脓肿。
  7. 痈：多个疖融合，在皮下脂肪、筋膜组织形成多个互相沟通的脓肿。
▶ 9. 风湿病病变发展各期病理变化

① 变质渗出期

结缔组织基质黏液样变性及胶原纤维发生纤维素样坏死，有浆液、纤维素和炎细胞渗出。

② 增生期（肉芽肿期）

细胞增生形成风湿小体（Aschoff body），其中为纤维素样坏死灶，周围有风湿细胞增生及少量淋巴细胞、单核细胞浸润。风湿细胞纵切面呈毛虫状，横切面如枭眼状。

③ 纤维化期

风湿小体发生纤维化，形成梭形瘢痕。
